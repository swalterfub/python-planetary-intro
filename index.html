#### Python overview - Interpreted, high-level general-purpose programming language - First released in 1991 - Emphasises code readability and simplicity by use of significant **whitespace** - Multiple programming paradigms: structured, **object-oriented** and functional - Comprehensive standard library, large collection of additional libraries* - numpy, gdal, matplotlib - Open!   ##### Syntax - No brackets to delimit blocks - Instead, whitespace indentation (visual structure represents semantic structure) for x in range(0, 3): print(x) ##### Object-oriented - *Objects* can contain data **and** methods myList = [1,2,3] myList.append(4) Note: List is a compound data type
#### Installation ##### Conda package and environment manager (1) Download and install miniconda for your operating system https://docs.conda.io/en/latest/miniconda.html (Python 3.X) (2) Start a conda terminal and install modules: * create a stand alone environment conda create -n tutorial * activates the environment conda activate tutorial * install modules needed for this tutorial: conda install -c conda-forge gdal jupyter matplotlib scipy rasterio * start Jupyter: jupyter notebook
