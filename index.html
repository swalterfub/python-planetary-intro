#### Installation ##### Conda package and environment manager (1) Download and install miniconda for your operating system https://docs.conda.io/en/latest/miniconda.html (Python 3.X) (2) Start a conda terminal and install modules: * create a stand alone environment conda install -n tutorial * activates the environment conda activate tutorial * install modules needed for this tutorial: conda install -c conda-forge gdal jupyter matplotlib rasterio * every time you start up a new terminal remember to type: conda activate tutorial
